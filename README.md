# Install

- <https://sanographix.github.io/illustfolio4/docs/>

# For developers

## Set Up

    $ npm install
    $ gulp

`/build` 以下に出力される

## 開発

- gh-pages 開発時、 `gh-pages` ブランチでは作業せず `gh-pages-dev` を使うこと
